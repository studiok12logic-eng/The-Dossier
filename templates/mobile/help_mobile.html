システムの概要

The Dossier（ドシエ）は、特定の対象人物に関する情報を体系的に収集・管理するためのプロトコルダー・アプリケーションです。 日々の些細な気づきから、戦略的な質問への回答まで、あらゆる断片を統合し、対象への理解を深めることを目的としています。

ターゲット管理

調査対象となる人物を新規登録し、基本情報（誕生日、住所、役割ランクなど）を一元的に管理します。

諜報ログ (Log)

タイムライン形式で情報を蓄積します。日々の言動、目撃情報、共有された秘密などをリアルタイムで記録可能です。画像添付にも対応しています。

戦略的質問 (Questions)

「オフィシャル」「ライフスタイル」などのカテゴリー分けされた質問に回答することで、ターゲットの深層を体系的に解明します。

User Ranks // 権限レベル

MASTER

最高特権

すべてのシステム設定、ユーザー管理、グローバル質問の編集権限を持ちます。

AGENT

標準諜報員

ターゲットの管理、ログの作成、質問への回答が可能です。自身のデータの編集権限を持ちます。
