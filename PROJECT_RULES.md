AIへの指示
開発における指示をするが下記のことを必ず遵守する事。

■fixした項目やデザインは変えない。
他と整合性を取るためやデータを統合する際に調整することはあるが、ユーザーが注文していた仕様は遵守する。

■このデータはAIが勝手に書き換えてはならない。

■下記に変えてはいけない仕様を随時記入していくので必ず遵守する事

〇共通
・HTMLのエンコードはUTF-8
・項目は日本語（簡単な英単語は使用してよい）

〇グループ
▼グループ項目
・グループの項目に接触頻度項目を追加、月火水木金土日をチェックする。

〇ターゲット
▼新規追加・編集ページ
・プロフィール画像
画像サイズは300×300pxとし、id=id_avatarに表示する際もそのsizeで。
アバター画像の削除ボタンを設置。

・生年月日
年を選択する際にoptionには
「yyyy年（干支）xx歳」
と表記
例：「1946年（戌）79歳」
年齢は数え年計算とする。
id=ageDisplayの年齢は月がわかれば月初めに加齢計算。日までわかれば正確な年齢を表示
id=etoDisplayの干支に英語表記は不要　例「干支: 戌」

・星座
name="zodiac_sign"に表示する星座は日本語で。
name="zodiac_sign"の下に誕生日の範囲を表示。
生年月日の月と日が入力されたら自動で自動判定に切り替える。

・記念日
削除はチェックボックスではなくボタンとする。
記念日を追加もボタンとし、押下した場合その場で項目を増やす。

▼ターゲットリスト

〇ダッシュボード
▼Target: Priority Alpha

〇諜報ログ
▼ターゲット選択画面
・画面をに分割し、左がターゲット選択画面。右がドシエ作成・編集画面とする。
・入力日の曜日が接触頻度項目にチェックされているグループのユーザーをアバターと名前・ニックネームのみのアイコン表示。
・選択画面の下部に追加ボタンを設置し、今日遭遇したターゲットをターゲット選択画面に追加。

▼ドシエ作成・編集
・発生日（基本は入力日）
・接触フラグ
・入力tab
　・出来事
　　入力欄
　　タグ
　・質問
　　質問カテゴリ選択
　　質問選択
　　回答入力

▼TARGET SELECT
defaultは当日でいいが、別の日にちを選択できるようにする。
選択した日にちで会うグループのターゲットと、addしたtarget（一度addしたtargetは何も記録してなくても追加し、ページを閉じても消えない）、誕生日orカスタム記念日が当日のtargetを表示する。

ターゲットselectのリストにマイナスアイコンを追加。押下するとターゲットselectから削除する。
記念日が同日のtargetのリストに記念日アイコンを表示する。
未記入のリストに未記入アイコンを表示する。
リストにニックネームの下に姓名と年齢を表示する。
例：田中　太郎（36）


〇左サイドメニューにカレンダーを追加。
誕生日・カスタム記念日を表記、出来事・質問・接触の入力があったかどうかの印も表示。
日付をクリックするとその日の諜報ログに飛ぶ。
