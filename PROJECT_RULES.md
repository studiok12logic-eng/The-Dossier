AIへの指示
開発における指示をするが下記のことを必ず遵守する事。

■fixした項目やデザインは変えない。
他と整合性を取るためやデータを統合する際に調整することはあるが、ユーザーが注文していた仕様は遵守する。

■このデータはAIが勝手に書き換えてはならない。

■下記に変えてはいけない仕様を随時記入していくので必ず遵守する事

〇共通
・HTMLのエンコードはUTF-8
・項目は日本語（簡単な英単語は使用してよい）

〇グループ
▼グループ項目
・グループの項目に接触頻度項目を追加、月火水木金土日をチェックする。

〇ターゲット
▼新規追加・編集ページ
・プロフィール画像
画像サイズは300×300pxとし、id=id_avatarに表示する際もそのsizeで。
アバター画像の削除ボタンを設置。

・生年月日
年を選択する際にoptionには
「yyyy年（干支）xx歳」
と表記
例：「1946年（戌）79歳」
年齢は数え年計算とする。
id=ageDisplayの年齢は月がわかれば月初めに加齢計算。日までわかれば正確な年齢を表示
id=etoDisplayの干支に英語表記は不要　例「干支: 戌」

・星座
name="zodiac_sign"に表示する星座は日本語で。
name="zodiac_sign"の下に誕生日の範囲を表示。
生年月日の月と日が入力されたら自動で自動判定に切り替える。

・記念日
削除はチェックボックスではなくボタンとする。
記念日を追加もボタンとし、押下した場合その場で項目を増やす。

▼TARGET SELECT
defaultは当日でいいが、別の日にちを選択できるようにする。
選択した日にちで会うグループのターゲットと、addしたtarget（一度addしたtargetは何も記録してなくても追加し、ページを閉じても消えない）、誕生日orカスタム記念日が当日のtargetを表示する。

ターゲットselectのリストにマイナスアイコンを追加。押下するとターゲットselectから削除する。
記念日が同日のtargetのリストに記念日アイコンを表示する。
未記入のリストに未記入アイコンを表示する。
リストにニックネームの下に姓名と年齢を表示する。
例：田中　太郎（36）


〇左サイドメニューにカレンダーを追加。
誕生日・カスタム記念日を表記、出来事・質問・接触の入力があったかどうかの印も表示。
日付をクリックするとその日の諜報ログに飛ぶ。


1. Accountsアプリ: ユーザー権限 (Roles)
    accounts/models.py の CustomUser に role フィールドを追加。
    階級定義 (Choices):
        MASTER: 管理者・運営 (全権限)
        ELITE_AGENT: 上級会員 (有料枠想定)
        AGENT: 一般工作員 (無料枠想定)
    デフォルト値は AGENT とする。
2. Intelligenceアプリ: 質問管理 (Shared vs Individual)
    intelligence/models.py に Question モデルを作成（または修正）。
    フィールド構成:
        user: ForeignKey (作成者)。
        is_shared: Boolean (デフォルトFalse)。MASTERのみがTrueにできる。
        content: 質問内容 (Text)。
        order: 表示順 (Integer)。
    ロジック要件:
        共有質問 (Shared): is_shared=True のレコード。全ユーザーの画面に表示される。編集・削除はMASTERのみ可能。
        個別質問 (Individual): is_shared=False のレコード。作成した本人(user)のみが表示・編集可能。
        取得クエリ: ユーザーが質問一覧を見る際は、「共有質問」OR「自分の個別質問」を結合して取得すること。

〇質問リスト
▼項目を追加
質問をカードにして並べる。　使用はターゲットリストに合わせる）
カードには詳細・編集・削除ボタン
質問・説明・回答形式・回答数・ポイント　をカードに表示
質問登録ボタンを設置。

▼質問登録項目
・質問分類
　共通・個別　どちらかを選択する。
・質問カテゴリー（追加方法はグループと同じ仕様で追加。）
　カテゴリー名・説明　単一選択
・質問ランク（追加方法はグループと同じ仕様）
　ランク名・ポイント　単一選択
・質問名
・説明（質問の意図）
・質問例（どんなふうに聞くかの具体例）
・回答方法
　選択式・自由記述（ロングテキスト）
・選択肢（選択式の場合のみ選択肢を追加できる。カンマ区切りで記入。タグと同じ仕様にする）
